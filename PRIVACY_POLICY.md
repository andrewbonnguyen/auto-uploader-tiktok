# Privacy Policy

Last updated: 2026

AutoUploader TikTok uses TikTok OAuth to authenticate users.

## Information Collected

- OAuth access token
- Basic user profile information

## Usage

Used only to:
- Authenticate user
- Upload video requested by user

## Data Storage

We do not permanently store:
- Passwords
- Videos
- Access tokens

## Third Parties

No data is shared.

Users may revoke access anytime in TikTok settings.
